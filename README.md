# EcommerceApi
This project is a RESTful API developed for the E-commerce platform. It provides endpoints for managing product catalog data, implementing robust error handling, and following professional Java development standards.

**API Endpoints**
HTTP Method | Path | Description
GET | /api/v1/products | Retrieve all products in the catalog
GET | /api/v1/products/{id}| Retrieve specific product details by ID

**Prerequisites**
Java: Version 25 (or your current JDK)
Build Tool: Gradle

**Setup Instructions**
1. Clone the repository: git clone https://github.com/zanjoebido/EcommerceApi.git
cd EcommerceApi
2. Run the application: ./gradlew bootRun
3. Testing:
    - The application will start at http://localhost:8080.
    - Use an API client like Thunder Client or Postman to send requests to the endpoints listed above.
